- 4 (quatro) espaços para indentação
- Espaço entre `if`, `while`, `for` etc e `(`: `if (ele > 180) {`
- { na mesma linha
- switches são (assim)[1]:
```
switch (var) {
case 1:
    // alguma coisa
case 3:
    // outra coisa
default:
    // mais uma coisa
}
```

[1]: http://www.oracle.com/technetwork/java/javase/documentation/codeconventions-142311.html#468
